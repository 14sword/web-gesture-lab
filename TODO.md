# 路线图 (Roadmap)

本项目未来的迭代升级与功能规划：

## 🚀 已完成阶段：基础建设与单手交互 (Phase 1)
*（当前版本特性，详见 [README.md](README.md) 和 [CHANGELOG.md](CHANGELOG.md)）*
- [x] 基于 MediaPipe 的手势核心坐标映射
- [x] 高性能单路径批量绘制 Canvas 渲染器
- [x] 星尘漩涡与粒子圣诞双核心互动模式
- [x] 全屏毛玻璃加载引导与设备自适应降级

---

## 🔮 阶段二：双手“双剑合璧”与流体力学 (Phase 2)

### 双手互动机制
- [ ] **物理吸引与斥力极点**：支持一只手做 `attract`（引力点），另一只手做 `stream`（斥力点），星尘粒子在这两个极点之间产生复杂的流线和轨迹运动。
- [ ] **合璧大爆发**：检测到双手握拳并靠拢时，粒子密度和光泽激增，触发范围大爆炸效果（Boom），将尘埃炸散至全屏，随后慢慢恢复。

### 新增特效模式
- [ ] **全景极光模式 (Aurora Borealis)**：引入平滑的曲线流动，手势控制极光的颜色、起伏频率和波动速度。
- [ ] **黑洞吞噬模式 (Black Hole Event Horizon)**：中心生成常驻的黑洞引力场，粒子被不断吞噬并从边缘重新随机生成，手势可以拨动黑洞吸积盘的朝向和大小。

---

## ⚡ 阶段三：性能与架构升级 (Phase 3)

- [ ] **WebGL 渲染器重构**：若粒子数量增加至 10,000 以上，将 Canvas 2D 渲染改写为 PixiJS 或 Three.js Shader 渲染，实现超大规模发光粒子表现。
- [ ] **手势分类判定鲁棒化**：引入更加平滑的滤波器（如一阶低通滤波/卡尔曼滤波），减少手势检测过程中的抖动和瞬态丢包。
- [ ] **音频发生器联动 (Synthesizer)**：根据手指在屏幕上的坐标实时改变音效合成器的主音频段和调制频率，让空中交互不仅能看，更能如同乐器般“弹奏”音色。

---

## ⚙️ 阶段四：辅助工具与自然语言自定义 (Phase 4)

- [ ] **自定义手势热键绑定 (Gesture Mapping Control)**：提供轻量级控制台，允许用户自行将“握拳”、“单指”等映射到不同的交互模式或场景变化上。
- [ ] **自然语言手势生成器 (Natural Language Builder)**：**【核心愿景】** 允许用户通过对话（例如“生成一个可以用双指捏合控制的进度条”），由 AI 动态生成并挂载 JSON UI 映射层，彻底解放界面定制难度。
- [ ] **完全本地化打包 (Electron Desktop app)**：将本项目使用 Electron 打包为独立的本地可执行桌面应用，默认集成所有的 WASM 文件和音频依赖。
